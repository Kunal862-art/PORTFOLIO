# Kunal Bansal | Portfolio 🚀

Welcome to the repository for my personal portfolio website!

## Overview
This is a modern, fully responsive, single-page portfolio website built to showcase my skills, projects, and achievements. It was designed from scratch without any heavy CSS frameworks to ensure maximum performance and customizability.

## Features ✨
- **Dark/Light Theme Toggle**: Seamless transition between dark and light modes using CSS variables and local storage.
- **Glassmorphism Design**: Modern frosted-glass aesthetic applied to cards and components.
- **Custom Animations**: Smooth scroll-triggered reveals using AOS, dynamic typing effect using Typed.js, and an interactive particle background using Particles.js.
- **Responsive Layout**: Optimized for desktop, tablet, and mobile viewing.
- **Universal Contact Form**: Form submission powered by Formsubmit.co.
- **Easter Egg**: Try entering the Konami Code (`Up, Up, Down, Down, Left, Right, Left, Right, B, A`) on the live site!

## Tech Stack 💻
- **HTML5**
- **CSS3** (Vanilla)
- **JavaScript** (Vanilla)
